# Git Commit Message Guidelines

Create commit messages with bullet points of all changes made in the session.

## Format
- Start with a summary title
- Follow with bulleted list of changes
- Include file names and specific modifications
- Be concise but descriptive
- **Use emojis for each bulletpoint**

## Example
```
Update authentication flow

✨ Added biometric login support
🔧 Fixed token refresh logic in auth_service.dart
📝 Updated README with new auth documentation
🐛 Resolved race condition in session management
```
